Informacion

Las etiquetas FORM e INPUT son la base de la interactividad en la web, permitiendo al usuario enviar información al servidor. El atributo type del input define qué clase de dato se espera (texto, contraseña, email, etc.). Algunos de los types para INPUT mas comunes son:
  • Text
  • password
  • Email
  • Number
  • checkbox
  • Radio
  • Submit
  • Date
  • Time
  • Range
