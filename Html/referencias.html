Pseudo-clases de paridad

Even - #even *:nth-child(even)

Selecciona automáticamente todos los elementos que se encuentran en posiciones impares (1, 3, 5, etc.) dentro de una lista o tabla.

Odd - #odd *:nth-child(odd)

Selecciona automáticamente todos los elementos que se encuentran en posiciones pares (2, 4, 6, etc.), ideal para crear tablas con colores de fila alternados.

Pseudo-clases de posición

First - Child - #first div:first-child

Selecciona y da estilo únicamente al primer elemento hijo que se encuentra dentro de su contenedor padre.

Only - Child .sho div:only-child

Aplica el estilo a un elemento siempre y cuando sea el único hijo disponible dentro de su contenedor padre (es decir, que no tenga hermanos).

Last - Child - #last div:last-child

Selecciona y da estilo únicamente al último elemento hijo que se encuentra dentro de su contenedor padre.

Nth - Child - #nth div:nth-child(5)

Permite buscar y seleccionar un elemento específico basándose en su posición numérica exacta dentro de su contenedor (por ejemplo, el hijo número 2 o número 5).

~

Aplica estilos a todos los elementos hermanos que se encuentran en el código después de un elemento específico, siempre y cuando compartan el mismo contenedor padre.

>

Selecciona y aplica estilos únicamente a los elementos que son descendientes de primer nivel (hijos directos) de un contenedor. A diferencia del selector de espacio tradicional, este ignora a los elementos que están anidados más profundo (los "nietos").

+

'
Selecciona el elemento que se encuentra inmediatamente después (el vecino directo) de otro elemento especificado en el código HTML, siempre y cuando ambos compartan el mismo contenedor padre.
